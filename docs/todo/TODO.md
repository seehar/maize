# Maize 功能待办清单

## ✅ 已完成

- [x] **中间件系统** - 下载器/爬虫/管道三层中间件链
- [x] **去重系统** - Classic 中间件去重 + Lite 内置去重
- [x] **Lite 轻量爬虫** - 内置并发/重试/代理/去重/深度控制/优先级/优雅关闭
- [x] **多下载器** - aiohttp / httpx / playwright / patchright
- [x] **多管道** - MySQL / Redis / CSV / 自定义
- [x] **RPA 爬虫** - Playwright/Patchright 浏览器自动化
- [x] **TaskSpider 任务爬虫** - 分批获取任务，数据库/API/Redis 任务源
- [x] **暂停/继续** - 爬虫运行时暂停和恢复
- [x] **基础分布式** - 基于 Redis 的分布式爬虫
- [x] **灵活配置** - 代码/配置文件/环境变量/YAML/TOML
